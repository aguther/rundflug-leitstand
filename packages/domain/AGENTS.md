# Domain-Paket

- Dieses Paket bleibt frei von Cloudflare-, HTTP-, Browser-, UI- und Datenbankabhängigkeiten.
- Zustandsübergänge werden als reine Funktionen modelliert.
- Jede neue Invariante benötigt mindestens einen Positiv- und einen Negativtest.
- Keine Sicherheitsentscheidung oder Freigabesemantik in Fachmodellen abbilden.
