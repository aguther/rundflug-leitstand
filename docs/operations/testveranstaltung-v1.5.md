# Synthetische Testveranstaltung V1.5

## Anlegen

1. In **Administration → Sicherung & Reset** „Neue Veranstaltung“ wählen.
2. Eine technische ID wie `test-flugtag-2026` und ausschließlich synthetische Angaben verwenden.
3. Für eine leere Demonstration „Vollständig neu einrichten“, für schnelle Vorführungen
   „Betriebsdaten zurücksetzen“ wählen.
4. Zwei Produkte, eine Ressourcengruppe, zwei Flugzeuge und zwei anonyme Pilotencodes anlegen.
5. Drei Gruppen der Größen 2, 1 und 1 verkaufen. Damit lassen sich Einzelgruppe, 2+1 und 1+1
   vorführen.

## Löschen

1. Optional Tagesbericht und Leistungsprofil exportieren.
2. In der Veranstaltungsliste **Löschen** wählen.
3. Exakte technische ID und kurze Begründung eingeben.
4. Ist es die letzte Veranstaltung, erscheint anschließend die Ersteinrichtung.

Die Testveranstaltung darf keine echten Gäste, Telefonnummern, Ticketcodes aus dem Betrieb oder
produktive Zugangsdaten enthalten.
