# Bedienkonzept: Veranstaltungsauswahl und Displaybindung

Status: vom Auftraggeber am 18.07.2026 freigegeben.

## Angemeldeter Arbeitsplatz

1. Konto auswählen und PIN eingeben.
2. Veranstaltung anhand von Name, Datum, Flugplatz und Status auswählen.
3. Den gewählten Veranstaltungstag ausdrücklich mit „Veranstaltung öffnen“ bestätigen.
4. Die Veranstaltung bleibt im Anwendungskopf sichtbar. „Veranstaltung wechseln“ führt zurück zur
   Auswahl, ohne die Anmeldung oder den fachlichen Zustand zu verändern.

Eine unbekannte, archivierte oder nicht mehr verfügbare gespeicherte ID wird nicht geöffnet. Die
Auswahl erscheint erneut. Bei genau einer Veranstaltung ist diese vorausgewählt, muss aber weiterhin
bestätigt werden.

## FIDS- und Gate-Anzeige

Die Administration erzeugt für ein technisches Anzeigegerät einen einmaligen QR-Code. Vorher werden
festgelegt:

- technische, nicht personenbezogene Bezeichnung,
- Veranstaltung (durch den aktuellen Administrationskontext),
- optionales Gate oder „gesamte Veranstaltung",
- Standardprofil (Deutsch) oder Terminalprofil (Englisch).

Das Anzeigegerät bestätigt die Kopplung einmalig und öffnet danach das FIDS im Kioskmodus. Ein
Widerruf oder eine neue Zuordnung erfolgt über die vorhandene Geräteverwaltung. Ein nicht
konfiguriertes Display zeigt nur den Einrichtungshinweis und keine synthetischen Daten.

## Mehrere Veranstaltungen

Operative Konten wählen den Veranstaltungstag je Anmeldung beziehungsweise bewusstem Wechsel.
Jedes feste Display besitzt dagegen genau eine lokale Bindung. Dadurch können mehrere FIDS-Geräte
gleichzeitig unterschiedliche Veranstaltungen oder Gates zeigen, ohne dass das Kioskpersonal beim
Start eine Auswahl treffen muss.
