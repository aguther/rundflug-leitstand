# Freigegebenes Gruppenticket-Konzept 1.8.0

Eine öffentliche Buchungsgruppe G erhält einen gemeinsamen Ticketzettel und genau einen QR-Code.
Der QR-Code öffnet `/gruppe/:code`. Bei einer nicht geteilten Gruppe wird ein Rundflug dargestellt;
bei bewusster Aufteilung erscheinen alle aktuellen Segmente als `Teilflug x von n` mit
Personenzahl, öffentlichem Status, Gate und absolutem Zeitfenster. Interne F-Kennungen und
individuelle Ticketcodes sind auf Zettel und öffentlicher Seite nicht sichtbar.

Zeitfenster werden in der Veranstaltungszeitzone als `ca. 14:20 – 14:40 Uhr` dargestellt. Unmittelbar
aufgerufene Teilflüge zeigen `Jetzt`, abgeflogene oder abgeschlossene Teilflüge `–`, unsichere
Prognosen `Wird aktualisiert`. Der Hinweis, dass es sich um eine Prognose und keine garantierte
Startzeit handelt, bleibt fachlich bestehen.
