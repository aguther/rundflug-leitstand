# Kassenkonzept 1.7.1

Release 1.7.1 ändert den Kassenablauf nicht. Das freigegebene Konzept aus
`v1.7.0-cashier-concept.md` gilt vollständig fort: ein gemeinsamer Gruppenzähler, feste kompakte
Produktzeilen, reservierte Aufteilungswarnung, systemgenerierte QR-Tickets, unveränderte Suche und
58-mm-Druckbild. Die neue einzeilige Kopfzeile gilt auch für die authentifizierte Kassenansicht;
FIDS bleibt ausgenommen.

Die Browserabnahme stellt sicher, dass diese Kopfzeilenänderung weder Produktaktionen noch
Ticketvorschau, Scan-Dialog, Suche oder Drucklayout beeinflusst.
